# YouTube Sentiment Analysis

A Python project that fetches YouTube comments and performs sentiment analysis using **TextBlob**, **VADER**, and a **Logistic Regression** model.

## Features

- Collect comments from any public YouTube video using the YouTube Data API.
- Analyze sentiment with:
  - TextBlob (simple lexicon-based analysis)
  - VADER (social media-tuned sentiment analysis)
  - Logistic Regression (machine learning model)
- Combine methods using an ensemble approach for better accuracy.
- Visualize sentiment distribution with bar charts and pie charts.
- Save analyzed comments to CSV for further study.
